# Odoo Dev

## Getting started

Requirements
  > - docker
  > - vscode
  > - install vscode recommended devcontainer extension `ms-vscode-remote.remote-containers`

Then, you should:
- clone this repo
- setup repositories (at least `odoo` repository into `src/community` folder)
- launch the dev container through the vscode extension.                                                 |
